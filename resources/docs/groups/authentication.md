# Authentication


## Log in




> Example request:

```bash
curl -X POST \
    "https://jobclass.bedigit.local/api/auth/login" \
    -H "Content-Type: application/json" \
    -H "Accept: application/json" \
    -H "Content-Language: en" \
    -H "X-AppApiToken: a25ydDlKdDRwT2wzYjAxV1hvc0hSUmQxYklTTE1pRHU=" \
    -H "X-AppType: docs" \
    -d '{"login":"user@demosite.com","captcha":"consectetur","password":"123456","captcha_key":"pariatur"}'

```

```javascript
const url = new URL(
    "https://jobclass.bedigit.local/api/auth/login"
);

let headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",

    "X-AppType": "docs",
};

    "login": "user@demosite.com",
    "captcha": "consectetur",
    "password": "123456",
    "captcha_key": "pariatur"
}

    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
```

## Log out

<small class="badge badge-darkred">requires authentication</small>



> Example request:

```bash
curl -X GET \
    -G "https://jobclass.bedigit.local/api/auth/logout/12" \
    -H "Authorization: Bearer {YOUR_AUTH_TOKEN}" \
    -H "Content-Type: application/json" \
    -H "Accept: application/json" \
    -H "Content-Language: en" \
    -H "X-AppApiToken: a25ydDlKdDRwT2wzYjAxV1hvc0hSUmQxYklTTE1pRHU=" \
    -H "X-AppType: docs"
```

```javascript

);

    method: "GET",
    headers,
}).then(response => response.json());
```
